Before you begin

This is an Android class. To complete the coding exercises, you will need an Android development environment with either an emulator or a device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

This course covers the fundamentals of touch interaction in Android. We look at how to programmatically respond to single-touch and multi-touch interactions.

Objectives

  1. Handle single-touch events
  2. Capture multi-touch interactions

Lecture

AND240 Touch

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Create a drawing application
    Exercise
    In this exercise, you will create a drawing app, which allows for a single finger to interact with the screen.
  2. MotionEventActions for multi-touch
    Exercise
    Here you will run a pre-built app to see the 5 different MotionEventActions that are reported for multi-touch events.
  3. Add multi-touch support to the drawing application
    Exercise
    You will update the drawing app to allow multiple fingers to interact with the screen simultaneously.
